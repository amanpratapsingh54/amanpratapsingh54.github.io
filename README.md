# Aman Pratap Singh — Portfolio (GitHub Pages)

This is a lightweight, recruiter-friendly portfolio built with plain **HTML/CSS/JS** and designed to be hosted on **GitHub Pages**.

## Files
- `index.html` — main page
- `style.css` — styling
- `script.js` — theme toggle, mobile menu, contact form mailto
- `assets/Aman_Resume.pdf` — resume file (replace anytime)

## Run locally
Just open `index.html` in a browser, or use a local server:

```bash
python3 -m http.server 8000
```

Then open: http://localhost:8000

## Deploy on GitHub Pages (recommended)
1. Create repo named: `<your-username>.github.io`
2. Push these files to the repo root
3. GitHub -> Repo Settings -> Pages
4. Source: Deploy from branch -> `main` + `/ (root)`

Your site will be live at: `https://<your-username>.github.io/`

## Add Google Analytics (optional)
In `index.html`, search for **Google Analytics (GA4)** and replace `G-XXXXXXXXXX` with your Measurement ID.
